## 二、新增页面数据契约

### 2.1 领榜教师端 — 资质认证

#### DGAP-01: 资质提交页

| 维度 | 说明 |
|------|------|
| **接口场景** | 教师提交资质认证材料 |
| **请求** | POST `/lb/teacher/certification/submit` |
| **请求字段** | `id_card_url`(string, 必填), `education_cert_url`(string, 必填), `teaching_cert_url`(string, 选填), `teaching_experience`(text, 选填), `real_name`(string, 必填) |
| **返回字段** | `certification_id`(string), `status`(enum: pending/rejected), `submit_time`(datetime), `review_deadline`(datetime) |
| **状态码** | 200 提交成功 / 400 参数缺失 / 409 重复提交(已有待审核) / 401 未登录 |
| **权限校验** | 教师角色 + 实名认证通过 |
| **数据校验** | 图片 url 有效 / real_name 2-20 字 / 提交次数 ≤ 3 |
| **前端展示** | 提交成功 → 跳转审核状态页 / 重复提交 → Toast 提示"已有待审核记录" |
| **后端处理** | 创建审核工单 → 通知后台审核队列 → 3次驳回冻结30天 |

#### DGAP-02: 资质审核状态页

| 维度 | 说明 |
|------|------|
| **接口场景** | 教师查看资质审核进度 |

| **请求字段** | - |

| **状态码** | 200 / 404 无记录 |
| **权限校验** | 教师角色 |
| **前端展示** | pending → 审核中动画 / approved → 认证标识展示 / rejected → 驳回原因+重提按钮 / frozen → 冻结倒计时 |
| **后端处理** | 查询审核工单最新状态 |

## 四、状态流转补充

### 4.1 教师资质认证状态机

```
未提交 ──(提交)──→ 待审核 ──(通过)──→ 已认证
                    │                    │
                    └──(驳回)──→ 已驳回 ──(重提)──→ 待审核
                                    │
                                    └──(3次驳回)──→ 已冻结(30天) ──(自动解冻)──→ 已驳回
```

### 4.2 投诉工单状态机

```
已提交 ──(分配)──→ 处理中 ──(处理完成)──→ 已解决 ──(确认)──→ 已关闭
  │                   │
  └──(超时48h)──→ 超时预警(标红)──→ 升级处理
```

### 4.3 提现状态机

```
待审核 ──(审核通过)──→ 待打款 ──(确认打款)──→ 已到账
   │
   └──(驳回)──→ 已驳回(解冻余额)
```
