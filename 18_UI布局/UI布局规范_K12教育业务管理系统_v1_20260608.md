## 五、响应式规则

| 设备 | 断点 | 布局策略 |
|------|------|---------|
| 手机竖屏 | < 768px | 单列/卡片布局/底部Tab |
| 手机横屏 | 768-1024px | 双列/紧凑卡片 |
| 平板 | 1024-1280px | 后台可用/侧栏折叠 |
| 桌面 | > 1280px | 后台标准布局/侧栏展开 |

小程序端不需要响应式适配（微信小程序固定手机屏幕）。后台按 1280px 最小宽度设计。
