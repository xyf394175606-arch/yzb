# 高保真原型说明：个人中心入口配置

## 背景

本说明对应 `10_高保真原型/高保真原型_个人中心入口配置_v1_20260623.html`。原型基于 `05_需求分析/需求分析_个人中心入口权限控制_v1_20260623.md` 生成，用于评审优坐标家长端、教师端个人中心 6 组入口展示配置。

## 目标

- 展示 PC 后台独立运营页如何统一配置个人中心入口展示。
- 展示家长管理、老师管理中通过资料区“权限控制”弹窗完成单用户入口独立控制。
- 展示移动端个人中心默认展示、入口隐藏态和历史链接不可访问状态。
- 通过右侧黄色需求标签承载页面规则、按钮结果、状态说明和待确认项。

## 关键结论

- 配置对象为 6 组入口：招募老师、申请加盟、邀请家长报名、招募合伙人、创客中心、会员开通。
- 普通会员、尊享会员两张卡分别支持独立展示控制。
- 创客中心关闭后隐藏团队卡、提现条和相关二级入口。
- 单用户入口控制优先于全局配置。
- 图片非必填，最终规格建议值待定。
- 保存配置后前后端立刻生效。

## 交互规则

- 点击左侧导航切换页面。
- 独立运营页点击“保存配置”打开保存确认弹窗。
- 保存确认弹窗点击“确认保存”后关闭弹窗，并展示“配置已保存，前后端立刻生效”状态。
- 独立运营页中“跳转类型”和“跳转目标”必须按行联动配置：固定页面对应页面选择，表单对应表单选择，外链对应 URL 填写。
- 家长端和教师端共用同一套入口文案、入口图片和跳转目标，但是否展示按家长端开关、教师端开关和单用户规则分别计算。
- 入口图片在配置表每行内处理，未上传也允许保存；图片规格建议值待定。
- 家长管理详情页按截图呈现左侧家长列表、中间家长资料、右侧客户跟进。
- 家长/老师资料区点击“权限控制”打开个人中心入口权限控制弹窗。
- 权限控制弹窗中每个入口支持继承全局、单独开启、单独关闭三种状态。
- 点击“保存权限”后关闭弹窗，并在当前用户详情页展示“权限配置已保存，前后端立刻生效”状态。
- 移动端入口关闭态只展示真实用户会看到的隐藏结果，不把规则说明放进手机线框。
- 历史链接不可访问页展示“入口已关闭”，并提供返回个人中心按钮。

## 状态说明

| 状态 | 表现 |
| --- | --- |
| 默认展示 | 入口按配置展示，会员卡、圆形入口、创客中心模块可见 |
| 单用户关闭 | 当前用户不展示对应入口，即使全局开启 |
| 单用户开启 | 当前用户展示对应入口，即使全局关闭 |
| 创客中心关闭 | 团队卡、提现条、相关二级入口均隐藏 |
| 历史链接关闭 | 直接访问入口详情时展示不可访问状态 |
| 图片未上传 | 允许保存，不阻断配置 |
| 权限弹窗保存 | 回到当前用户详情页，展示立即生效状态 |

## 权限差异

- 只有系统权限配置中具备入口配置权限的后台角色可以编辑。
- 无权限角色不可编辑全局配置或单用户独立控制。
- 家长端和教师端使用同一套文案、图片、跳转目标，但展示结果按端开关和单用户规则决定。
