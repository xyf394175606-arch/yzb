# 需求分析：优坐标课程售卖闭环

## 1. 背景与范围

本稿基于优坐标 Axure HTML 原型、`需求分析_优坐标Axure原型拆分_v1_20260629.md`、`需求分析_优坐标业务闭环模块拆分_v2_20260629.md` 和上游 MVP 需求分析整理，单独分析“课程售卖闭环”。本闭环解决课程从后台配置、前端展示购买、订单生成、支付状态、订单查看到评价入口的完整售卖问题。

目标是让后续业务流程、信息架构、交互设计和原型设计能独立处理课程售卖，不再从总文档中翻找字段和按钮。本稿只分析需求，不输出 API、数据库或开发实现，不重画原型。

输入来源包括：Axure 原型中的 `课程管理`、`购课订单`、`会员充值订单`、`我的`、`我的订单`、`评价管理`、历史参考 `原优化界面`；以及上游需求线索中关于优坐标课程/老师资源展示、会员/课包和排课对接的说明。

关键结论：

- 已确认需求：后台可配置课程基础信息、价格、课时、封面、介绍、关联老师；后台购课订单展示课程订单和试听课订单；试听课订单 0 元、无需支付、默认已完成。
- 推测需求：小程序当前课程详情和支付页可能需要从 `原优化界面` 回流，但当前有效页面树未完全证明。
- 待确认需求：课程详情页、支付页、待支付/已支付/已取消状态流转、支付超时规则。
- 暂不纳入范围：`购课用户列表（暂时不用）`、历史 `原优化界面` 中未确认回流的支付与课程页面。

## 2. 业务规则

- 后台 `课程管理` 是课程售卖闭环的配置起点，课程名称、分类/学科、适用年级、课程类型、价格、课时、封面、介绍、状态和关联老师用于支撑前端展示、下单、订单金额和后续排课。
- 课程价格包含原价、会员单价、平台服务费、课时数、单节时长和总价。会员用户前 100 课时按折扣价，超出部分按正价；普通用户全部课时按正价计费。该计价规则来自原型说明，但 100 课时是否为正式规则仍需确认。
- 试听课金额固定为 0 元，无需支付，支付方式显示“无需支付”，订单状态默认“已完成”；购课订单中不新增单独订单类型，而是在总购买课时单元格标记“1节・试听课 0元”。
- `购课订单` 只展示课程订单和试听课订单，不展示会员充值订单；会员充值订单归属会员权益闭环。
- 购课订单列表当前只确认有“详情”操作，无退款、取消、编辑、删除操作。
- 课程支付完成后形成后续排课对象；订单或课节达到评价条件后进入评价闭环。
- 课程关联老师会影响课程展示、排课授课老师和教师详情评价记录，属于课程售卖与教师师资闭环的交叉点。

## 6. PC/移动端差异

PC 端：

- PC 后台负责课程配置、价格维护、关联老师、订单查询和评价管理。
- `购课订单` 当前只允许查看详情，不承担用户侧支付或订单取消。

移动端：

- 小程序负责课程展示、购买、订单查看和评价入口。
- 当前有效小程序页面中可确认 `我的`、`我的订单`，课程详情/支付页主要来自历史 `原优化界面`，是否回流待确认。

历史/暂废：

- `原优化界面` 可作为课程详情、支付、会员/非会员支付差异参考，不默认进入当前有效主链路。
- `购课用户列表（暂时不用）` 不纳入当前售卖闭环。

## 8. 页面与跳转逻辑

```text
后台课程管理配置课程/价格/关联老师
-> 小程序课程展示或历史课程详情页参考
-> 用户购买课程或试听课
-> 生成购课订单
-> 支付完成/试听课0元完成
-> 我的订单展示订单状态
-> 支付完成后进入排课上课闭环
-> 达到评价条件后进入评价闭环
```

## 9. 待确认问题

- 当前课程详情、课程支付、会员/非会员支付页面是否从 `原优化界面` 回流到当前小程序范围。
- 购课订单的待支付、已支付、已取消、已完成状态流转和支付超时规则。
- 试听课订单是否始终 0 元且默认已完成，是否存在取消、过期或重复领取限制。
- 会员用户前 100 课时按折扣价是否为正式规则。
- 课程关联老师后，前端展示老师与排课默认老师是否完全一致。
- 评价开放条件是订单支付完成、课节完成、签到完成，还是订单完成。

## 10. 风险与依赖

- 课程售卖依赖会员权益闭环提供会员身份、折扣课时和计价规则。
- 支付完成后的履约依赖排课上课闭环。
- 订单后的口碑反馈依赖评价闭环。
- 若历史课程详情和支付页不回流，当前小程序课程购买链路存在页面缺口。

## 11. 下一步动作

- 业务流程阶段补齐课程配置、购买、支付、订单完成、排课触发和评价触发流程。
- 信息架构阶段确认课程详情、支付页、订单详情是否为当前有效页面。
- 交互设计阶段补齐待支付、支付失败、取消、无订单、课程下架等状态。

## 12. 原型设计输入

- 页面范围：课程管理、购课订单、我的订单、课程详情/支付页待确认、评价入口。
- 字段批注：课程价格、会员价、平台服务费、折扣课时、正价课时、订单状态、试听课 0 元。
- 按钮批注：新增/编辑课程、关联老师、购买、订单详情、评价。
- 黄色说明区：会员/非会员计价、试听课 0 元、购课订单仅详情、历史页面是否回流待确认。
