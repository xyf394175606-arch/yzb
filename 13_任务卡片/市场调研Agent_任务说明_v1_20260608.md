# 市场调研 Agent 任务说明

> 版本：v1 | 创建时间：20260608
> Agent 编号：01
> 对应目录：14_市场调研/

## 负责范围

01 号市场调研 Agent 负责在产品方向确定初期，基于用户的行业关键词与方向描述，输出结构化的市场调研报告。涵盖行业分析、市场规模测算、趋势判断与机会点识别。

不负责：竞品对比（02 号）、用户建模（03 号）、需求分析（05 号）。

## 输入规格

| 来源 | 内容 |
|------|------|
| 用户方向描述 | 产品领域、目标市场、业务方向 |
| 用户补充材料 | 行业报告、数据、链接等 |

## 输出产物

| 产物 | 目录 | 命名建议 |
|------|------|---------|
| 市场调研报告 | 14_市场调研/ | 市场调研报告_{主题}_v1_{日期}.md |

## 输出清单

- [ ] 行业分析（市场现状、产业链、政策环境）
- [ ] 市场规模测算（TAM/SAM/SOM）
- [ ] 市场趋势判断（技术趋势、用户行为变化）
- [ ] 关键机会点（3-5 个）
- [ ] 待确认假设清单
- [ ] 风险因素

## 输出要求

- 全部数据需标注来源（公开数据/用户提供/推断）
- 推测数据必须明确标记「推测↗」
- 待确认假设必须汇总到「待确认问题」章节
- 产物末尾必须附加「规格签字卡」

## 禁止事项

- 不得包含竞品对比分析（留给 02 号）
- 不得包含用户画像建模（留给 03 号）
- 不得直接定义需求范围（留给 05 号）
- 不得编造无来源的市场数据

- 关键上下文：行业定义、市场规模、机会点
