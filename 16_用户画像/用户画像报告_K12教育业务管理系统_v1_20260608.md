## 二、用户旅程图

### 2.1 领榜家长核心旅程

```
[触发]  孩子成绩下滑 → 家长群看到推荐 → 打开领榜小程序
[探索]  浏览平台 → 了解会员权益 → 购买会员（或直接发需求）
[行动]  填写家教需求（年级/科目/时间/预算/位置） → 提交发布
[等待]  看到需求状态更新 → 老师付费接单 → 收到企微联系
[沟通]  与老师在企微沟通 → 确认上课细节
[服务]  上课 → 课后评价 + 上传图片/视频
[复购]  下次需要再发布新需求
```

| 阶段 | 情绪 | 痛点 | 机会点 |
|------|------|------|--------|
| 触发 | 😟→😐 | 选择困难 | 社交推荐 + 会员信任 |
| 探索 | 😐→😊 | 了解不足 | 清晰的权益说明 |
| 行动 | 😊 | 填表繁琐 | 模板化需求表单 |
| 等待 | 😐→😟 | 无人接单焦虑 | 接单状态实时通知 |
| 沟通 | 😐 | 信息不对称 | 教师资料透明化 |
| 服务 | 😊→😄 | 效果验证 | 课后评价体系 |
| 复购 | 😊 | 重新发布麻烦 | 一键复投需求 |

### 2.2 优坐标家长核心旅程

```
[触发]  朋友推荐 → 打开优坐标小程序
[了解]  浏览产品/老师 → 比较会员权益 → 购买卓越会员
[咨询]  联系客服 → 描述孩子情况 → 客服推荐老师和课程
[确认]  确认课程方案 → 支付 → 等待排课
[排课]  客服安排时间 → 确认上课安排 → 开始上课
[跟踪]  课后反馈 → 满意继续 / 不满意联系客服替换老师
```

### 2.3 领榜教师核心旅程

```
[注册]  认证身份 → 填写资料 → 上传资质
[浏览]  查看可接需求列表 → 筛选（科目/年级/区域/价格）
[决策]  评估需求质量 → 付费接单（3x 课时费）
[联系]  获取家长企微 → 主动联系 → 确认上课
[服务]  上课 → 记录服务过程
[收益]  家长评价 → 评分提升 → 更多接单机会
```
