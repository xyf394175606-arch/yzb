一、页面整体规则

本页面只展示课程订单 + 试听课订单，不单独展示会员充值订单。


二、列表字段说明（从上到下）

订单编号 / 用户信息 / 用户类型 / 课程名称 / 总购买课时 / 折扣课时 / 正价课时 / 单价 / 订单金额 / 支付方式 / 订单状态 / 下单时间 / 操作


三、业务规则标注

（1）用户类型

普通用户：全部课时按正价计费

会员用户：前 100 课时按折扣价，超出 100 课时部分按正价

（2）试听课特殊规则

试听课金额固定 0 元

无需支付，支付方式显示：无需支付

订单状态默认：已完成

不在单独加订单类型，在总购买课时单元格标记：1 节・试听课 0 元 灰色标签样式

折扣课时、正价课时都填 0

（3）价格规则

会员折扣价、课程正价由后台配置

自动计算：折扣课时 × 折扣价 + 正价课时 × 正价价 = 订单金额

试听课统一 0 元


四、操作功能标注

列表操作：仅详情

点击详情弹出订单详情弹窗，弹窗包含所有明细字段

无退款、无取消、无编辑、无删除
